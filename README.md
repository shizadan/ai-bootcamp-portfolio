### 3. SimShop Network — Business Management System
**Repo:** [simshop-network](https://github.com/shizadan/simshop-network)  
**Live App:** https://simshop.netlify.app  
**Tools:** HTML, CSS, JavaScript, Firebase, Firestore, Netlify

- Real-time stock, sales and revenue tracking across all branches
- Distribution logging and custom date range reporting
